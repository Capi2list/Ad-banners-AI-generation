# Ad-banners-AI-generation

## Установка

Для работы с main.py необходимо установить библиотеки из requirements.txt, версия python 3.10.13.

```
pip install -r requirements.txt
```

После этого можно запускать скрипт main.py, однако для работы скрипта требуется GPU с ~16 ГБ видеопамяти (Запускается на стандартной T4), иначе вся модель не сможет поместиться и инференс не будет запускаться. После запуска main.py скрипт выдаст ссылку на UI интерфейс градио, где можно опробовать модель. 

Обязательно входным изображением должна быть картинка 1024x1024, где явно выделен фон, желательно белый (примеры таких изображений можно посмотреть в Testing images, в прилагающейся ссылке на гугл диск). Гиперпараметры выставлены по умолчанию, их можно сменить при желании.  

## Dreambooth

Для работы с Dreambooth скрипт написан в ноутбуке training.ipynb, он запускает скрипт из [Kohay_ss](https://github.com/kohya-ss/sd-scripts) с нужными гиперпараметрами. Веса модели для тренировочного датасета, который использовался в отчете и презентации, можно увидеть в гугл диске.

## Metrics

Модель была протестирована при помощи LLM Gemini-3.1-flash-lite, результаты судьи для каждого изображения из подборки записаны в metrics.csv для итоговой модели, в metrics_default.csv - для стандартного SDLX. Там можно и увидеть пояснения модели к каждому случаю. 

|                      |   Result model |   Default SDLX |
|:---------------------|---------------:|---------------:|
| object_preservation  |           6.88 |           2.02 |
| prompt_adherence     |           7    |           1.94 |
| aesthetic_quality    |           7.58 |           6.42 |
| background_relevance |           7.06 |           6.82 |

object_preservation - сохранение продукта \
prompt_adherence - соответствие промпту \
aesthetic_quality - общее качество \
background_relevance - качество фона
